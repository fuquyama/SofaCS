# SofaCS
Unofficial C# porting of the International Astronomical Union's (IAU) "Standards Of Fundamental Astronomy" (SOFA) library official C release.

## Description
This SofaCS is unofficial porting from the International Astronomical Union's (IAU) "Standards Of Fundamental Astronomy" (SOFA) library official C release.

SOFA C ライブラリを C# に移植したものです. 

移植の完全性を証明するため、SOFA C のテストコードも C# に移植して完全再現しています.

## Package Version <-> SOFA Release Correspondence

The package versions correspond to the following releases of the SOFA C Library:

| Package Version | SOFA Release |
| --------------- | ------------ | 
| v1.0            | 2021-05-12   |
